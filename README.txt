txt-to-yaml is a text processing script for a particular format of text data.
the script reads the text file line by line.
converter.py parses the text from file requirements/a6061-t6.mat.txt and outputs YAML from the processed data.

Get the latest python from https://www.python.org/

remember to check the ADD TO PATH option from the installer window 

after installation open cmd/Terminal and enter the below command

pip install PyYAML pandas (for Windows)
pip3 install PyYAML pandas (for Linux and MacOS)

for running the script goto the directory using terminal or open the terminal in directory
enter
python converter.py

A file dailog will open to select the base txt file, select file and click ok
The output will be saved to output.yaml
